# 固定音声キャッシュ版

この版では、出勤時・退勤時の音声を固定文にしました。

- 出勤時: 「奥田さん、おはようございます。今日も一日、無理のないペースで頑張りましょう。」
- 退勤時: 「奥田さん、今日もお疲れさまでした。今日の作業はここまでです。ゆっくり休んでくださいね。」

## 仕組み

初回だけ `/api/tts` でGemini音声を生成し、生成した音声をブラウザの `localStorage` に保存します。
2回目以降は保存済み音声を再生するため、出勤・退勤ボイスはかなり速くなります。

## Vercel環境変数

OpenAIは不要です。Gemini版なので以下だけでOKです。

```txt
GEMINI_API_KEY=自分のGemini APIキー
```

## 文言を変えたい場合

`src/App.tsx` の `getFixedVoiceText` 内の文章を変更してください。
文章を変更した場合は、`FIXED_VOICE_CACHE_VERSION` を `v2` などに変えると、新しい音声で作り直されます。
