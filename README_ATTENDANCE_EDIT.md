# 打刻漏れ追加・打刻記録修正版

この版では、退勤タイマー画面に以下を追加しています。

- 「打刻漏れを追加」フォーム
  - 日付、出勤時刻、退勤時刻、メモを入力して後から保存できます。
  - 保存した記録には「手動追加」ラベルが付きます。

- 履歴の「修正」機能
  - 退勤活動ログの各記録から、日付・出勤時刻・退勤時刻・メモを修正できます。
  - 修正した記録には「修正済み」ラベルが付きます。
  - 修正前の出勤・退勤時刻も表示されます。

- ジョブカンボタン
  - 既存どおり「ジョブカンを開く」ボタンを残しています。

注意：この機能は自分用の勤怠メモです。会社の正式な勤怠修正は、ジョブカン側で行ってください。
